## 图表和文件输出要求

### ⚠️ 强制要求：中文字体设置

**所有使用 matplotlib 的 Python 代码，必须在导入 matplotlib 后立即设置中文字体！**

```python
import matplotlib
import matplotlib.pyplot as plt

# 🚨 这两行代码是强制要求的，必须包含！
matplotlib.rcParams['font.sans-serif'] = ['Arial Unicode MS', 'PingFang SC', 'Microsoft YaHei', 'SimHei', 'WenQuanYi Micro Hei']
matplotlib.rcParams['axes.unicode_minus'] = False  # 解决负号显示问题
```

**原因**：如果不设置中文字体，图表中的中文标签、标题、图例都会显示为方框（□□），无法正常阅读！

### 其他要求

1. **保存方法**：必须使用 `plt.savefig()` 或 `fig.savefig()`，禁止使用 `plt.show()`
2. **文件格式**：
   - 图像：.png, .jpg, .jpeg, .svg, .gif
   - 数据：.csv, .xlsx, .xls, .json
   - 文档：.txt, .md
3. **文件命名**：使用描述性名称，如 `sales_trend_2024.png` 而不是 `figure1.png`
4. **文件自动显示**：

   - **其他文件显示为可下载链接** - CSV、JSON等文件会显示为下载按钮

## 代码示例

```python
import matplotlib.pyplot as plt
import matplotlib
import numpy as np

# 设置中文字体支持（必须在其他导入后立即设置）
# macOS: PingFang SC, STHeiti
# Windows: Microsoft YaHei, SimHei
# Linux: WenQuanYi Micro Hei, DejaVu Sans
matplotlib.rcParams['font.sans-serif'] = ['Arial Unicode MS', 'PingFang SC', 'Microsoft YaHei', 'SimHei', 'WenQuanYi Micro Hei']
matplotlib.rcParams['axes.unicode_minus'] = False  # 解决负号显示问题

# 生成数据
x = np.linspace(0, 10, 100)
y = np.sin(x)

# 创建图表
fig, ax = plt.subplots(figsize=(10, 6))
ax.plot(x, y, label='正弦曲线', linewidth=2)
ax.set_xlabel('X轴', fontsize=12)
ax.set_ylabel('Y轴', fontsize=12)
ax.set_title('正弦函数', fontsize=14, fontweight='bold')
ax.legend()
ax.grid(True, alpha=0.3)

# 重要：保存图表文件
fig.savefig('sin_function.png', dpi=300, bbox_inches='tight')
plt.close(fig)  # 释放内存
print('✅ 图表已保存为 sin_function.png')
```
