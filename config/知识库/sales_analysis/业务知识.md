# 销售数据分析知识库

## 核心指标定义

### 销售额（Sales Revenue）
- 定义：企业通过销售商品或提供服务所获得的总收入
- 计算公式：销售额 = 单价 × 销售数量
- 相关维度：时间（日/周/月）、产品类别、销售渠道、地区
- 业务价值：衡量业务规模和盈利能力

### 订单量（Order Count）
- 定义：在一定时间内完成交易的订单总数
- 计算公式：订单量 = 订单ID去重计数
- 相关维度：时间、客户类型、订单金额区间
- 业务价值：反映市场需求和客户活跃度

### 客单价（Average Order Value）
- 定义：平均每笔订单的交易金额
- 计算公式：客单价 = 销售额 / 订单量
- 业务价值：评估客户消费水平和营销效果

### 转化率（Conversion Rate）
- 定义：访客转化为购买客户的比例
- 计算公式：转化率 = 下单用户数 / 访客数 × 100%
- 业务价值：衡量营销漏斗效率

### 复购率（Repurchase Rate）
- 定义：二次及以上购买客户占比
- 计算公式：复购率 = 复购用户数 / 总购买用户数 × 100%
- 业务价值：反映客户忠诚度

## 常见分析场景

### 1. 销售趋势分析
**目标**：了解销售数据随时间的变化趋势
**分析方法**：
- 日/周/月销售趋势对比
- 同比/环比分析
- 季节性规律识别
- 预测未来走势

**关键指标**：
- 销售额趋势
- 订单量趋势
- 客单价变化

**典型问题**：
- "最近一个月的销售额趋势如何？"
- "今年销售额与去年相比增长了多少？"
- "销售数据的季节性特征是什么？"

### 2. 产品表现分析
**目标**：评估不同产品的销售表现
**分析方法**：
- 产品销售额排名（TopN）
- 产品销售占比
- 产品生命周期分析

**关键指标**：
- 产品销售额
- 产品销售量
- 产品毛利率

**典型问题**：
- "销售额Top10的产品有哪些？"
- "每个产品类别的销售占比是多少？"
- "哪些产品销售下滑严重？"

### 3. 渠道表现分析
**目标**：对比不同销售渠道的业绩
**分析方法**：
- 渠道销售对比
- 渠道转化率分析
- 渠道ROI评估

**关键指标**：
- 渠道销售额
- 渠道订单量
- 渠道获客成本

**典型问题**：
- "线上和线下渠道的销售额对比如何？"
- "哪个渠道的转化率最高？"
- "各渠道的客户质量有什么差异？"

### 4. 客户分析
**目标**：深入了解客户群体特征
**分析方法**：
- 客户分群（新客/老客/VIP）
- 客户价值分析（RFM模型）
- 客户留存分析

**关键指标**：
- 新客数量
- 老客数量
- 客户生命周期价值（LTV）

**典型问题**：
- "新客和老客的消费行为有什么差异？"
- "高价值客户有哪些特征？"
- "客户复购率是多少？"

## 数据分析建议

### 时间维度选择
- **实时监控**：使用小时或日级别数据
- **趋势分析**：使用周或月级别数据
- **长期对比**：使用季度或年级别数据

### 空间维度选择
- **地区分析**：省、市、区县级
- **门店分析**：具体门店或区域
- **渠道分析**：线上/线下/具体渠道

### 聚合方式选择
- **求和**：适用于销售额、订单量
- **平均**：适用于客单价、转化率
- **计数**：适用于客户数、产品数

## 注意事项

1. **数据质量**：确保销售数据完整、准确、一致
2. **时间范围**：选择合适的时间范围，避免异常值干扰
3. **对比基准**：同比/环比时要考虑节假日、促销活动等因素
4. **多维交叉**：从多个维度交叉分析，发现深层洞察
